# Code of Conduct

This project follows a simplified Code of Conduct.

## Our pledge
We are committed to providing a welcoming and respectful experience for everyone.

## Expected behavior
- Be respectful and constructive
- Assume good intent
- Provide actionable feedback

## Unacceptable behavior
- Harassment or discriminatory language
- Personal attacks

## Enforcement
Project maintainers may remove or edit contributions that violate this code.
